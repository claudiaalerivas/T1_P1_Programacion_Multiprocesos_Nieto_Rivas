## Clase IsPrime

Esta clase determina si un número es primo.

![image](./images/image-7.png)

### FUNCIONES

- **isPrime(int number)**
Función boolenana que retorna TRUE o FALSE dependiendo si un número pasado como argumento es primo o no.
